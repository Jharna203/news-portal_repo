What's the difference among Var, Let, and Const?

var :

                                        var x = 1;

                                        if (x === 1) {
                                          var x = 2;
                                        
                                          console.log(x);
                                          // expected output: 2
                                        }
                                        
                                        console.log(x);
                                        // expected output: 2
var declarations are globally scoped or function/locally scoped. This means when var is declared within or outside a function, it is available and can be accessed within or outside that function block, also it is available for use in the whole window.

Let :

                                        let x = 1;

                                        if (x === 1) {
                                          let x = 2;
                                        
                                          console.log(x);
                                          // expected output: 2
                                        }
                                        
                                        console.log(x);
                                        // expected output: 1
if a variable declared in a block with 'let', it only available for use within that block. A variable declared with let can be updated within its scope.

const :

                                        const number = 42;

                                        try {
                                          number = 99;
                                        } catch (err) {
                                          console.log(err);
                                          // expected output: TypeError: invalid assignment to const `number'
                                          // Note - error messages will vary depending on browser
                                        }
                                        
                                        console.log(number);
                                        // expected output: 42
Like let declarations, const declarations can only be accessed within the block they were declared.It cannot be updated or re-declared.

What's the difference between Regular Functions and Arrow Functions?

Regular Functions :

                                    const square = function(x){
                                        return (x*x);
                                      };
                                      console.log(square(9));
Regular functions created using function declarations or expressions are constructible and callable. Since regular functions are constructible, they can be called using the new keyword.Arguments objects are not available in arrow functions, but are available in regular functions.Unlike regular functions, arrow functions do not have their own this.

Arrow Functions :

                                        const square = (x) => {
                                            return (x*x);
                                        };
                                        console.log(square(9));
Arrow functions avoid a lot of code scenarios that can lead to misunderstandings, it makes our code clearer and more controllable.Curly brackets arenâ€™t required if only one expression is present. Why do we use backticks or Template strings in JavaScript?

Template strings :

                                        const longString = `At vero eos et  provident, similique sunt in deserunt animi.`
Template strings are a powerful feature of modern JavaScript released in ES6.This is much better than the alternative in old JavaScript where we had to concatenate a string. It allows us to create strings that are complex and contain dynamic elements. Another great thing that comes with template strings are tags. Tags are functions that take a string and the decomposed parts of the string as parameters and are great for converting strings to different entities. We can use it for Interpolating Variables and Expressions, Multiline Strings, Nesting Templates,Tagged Templates, Raw Strings.
